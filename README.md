# Analisis_Econometrico_Inflacion_MX
Análisis Econométrico: La Dinámica Monetaria de la Inflación en México (2010-2025)

Objetivo: Validar empíricamente la teoría cuantitativa del dinero de Milton Friedman utilizando datos reales del Banco de México y modelado estadístico en Python.

Stack Tecnológico: Python, Pandas, Statsmodels, Seaborn, Matplotlib.

Metodología:

Extracción de datos financieros (M1, M2, INPC, Tipo de Cambio).

Transformación de series de tiempo y cálculo de tasas de crecimiento anual.

Modelado de Regresión Lineal Múltiple (OLS) y análisis de significancia.

Análisis de Rezagos Temporales (Lag Analysis) para determinar tiempos de transmisión.

Diagnóstico de residuos (Pruebas de Normalidad).

Hallazgos Clave:

La inflación en México es un fenómeno monetario con un rezago de transmisión de 12 meses.

El dinero transaccional (M1) tiene un impacto inflacionario 3 veces mayor que la oferta monetaria amplia (M2).

El modelo final explica la dinámica de precios con robustez estadística (residuos normales).
